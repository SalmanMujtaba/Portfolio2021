Angular-5 Contact Form

The Contact form as the name suggests is used to send an e-mail to the owner. The form, to an extent, is serverless, as it uses Firebase’s Cloud function to send the e-mail.

Features

Uses Angular 5’s form validation techniques to validate the form. It is serverless in the sense that it does not require a server explicitly. Firebase’s Cloud Function and NodeMailer help in sending the email whenever data is added to the Firebases Real Time Database.

Technology

Crafted the page using my favorite Google’s Material Design. Other quirks were realized using TypeScript. AngularFire2 library connects Angular to Firebase.

Back-End

Firebase helps in creating Cloud Functions which act as a Trigger, therefore, whenever data is added to its database, the Cloud Function, with the help of Nodemailer sends the email to the owner. This, of course, happens asynchronously.
