Picklebook

A comprehensive booking system for pickleball. Manage court bookings, player registrations, and session coordination across multiple groups.

Features

Picklebook is a court booking system that helps users book courts or join the waitlist. It sends updates using Resend for important updates. Picklebook also allows the admin to manage private groups that determine the session visibility. Users can see the other session players.

Code

The Picklebook app was built using Next.js with TypeScript and Tailwind CSS. It utilizes Firebase for authentication and Firestore for storing session data. The app follows a clean architecture and utilizes React Context API for state management.

Libraries

Picklebook uses various libraries including @radix-ui, Lucide React, React Icons, and Resend. The app also utilizes the Shadcn/ui library for UI components.

Back-End

Picklebook stores session data in Firebase's Firestore. The app utilizes Firebase Authentication for user authentication and management.
