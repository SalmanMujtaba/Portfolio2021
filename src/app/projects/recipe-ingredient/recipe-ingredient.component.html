Recipe Ingredient Angular App

The Recipe-Ingredient, Angular 5 based app is used to manage Recipes and Ingredients pertaining a User’s account. The user is authenticated using Firebase.

Features

It enables a user to register and then log in to their account, where they are able to add or modify Recipes, push the ingredients to a shopping list and modify the shopping list. The user data is persisted after logging out. All the data is stored in Firebase. In addition to the under the hood features, I utilized Angular 5s feature new form validation and Routing techniques.

Technology

The app is based on Typescripts where the UI is based on Google’s Material Design. Certain features were added later on after the introduction of Angular 5. Several components were created which communicated in various ways. Extensively used Event Emitters, Property Binding, Decorators, Pipes, Dependency Injections, Directives and Observables.

Configuration

Firebase was setup to enable communication with the database. As part of the future scope, I'll implement Redux and host the application.
