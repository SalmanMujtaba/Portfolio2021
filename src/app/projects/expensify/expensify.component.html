Expensify App

The is a Single Page Application, based on React JS, where a user can keep a track of their expense. It makes use of Firebase and Redux to maintain the data of the user, which persists after the user logs out.

Features

It’s a Single Page Application which leverages Reacts Routing. Only authenticated users are allowed to use the app. Reduxs’ components: Actions, Selectors and Reducers along with Firebase facilitates user’s data management. Incorporated airbnb react routing to act as a Date Picker for Expenses.

Code

The App uses JSX and ES 6 features extensively. Some of the ES 6 features used are: Object Structing/Destructuring, Arrow functions, Spread Operator etc.

Configuration

Configured Webpack to support SCSS files etc. Optimized the project files for production level. Used react-dates for selecting dates.

Back-End

Asynchronous calls to the Firebase’s database using Redux modifies the user’s data. Firebase also expedites user authentication.

Styling

The app is styled using SCSS which is a great way to write modular CSS. I used the naming convention as prescribed in BEM: Block Element Modifier.
