GitHub Repository Search

The app enables the users to search for the Jvascript GitHub repositories, and their corresponding recent commits.

Features

The user can search for JS repositories present in the GitHub. The application also shows commits of the last 30 days of corresponding repository. A simple form vaildation is in place. It can be viewed from here. The error dialog shows an intuitive error in case of any API failure.

Technology

The app is built using Angular, and Googleâ€™s Material Design. It makes use of the Neomorphic design pattern for styling. The GitHub APIs along with the token is used to fetch the data.
