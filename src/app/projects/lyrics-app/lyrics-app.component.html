Lyrics App

The App is based on React JS and GraphQL. The user has the ability to add a Song, add a Lyrics to the song, delete a Song, delete a Lyrics and Like a Song. It is a Single Page Application which uses React-Routing for navigation..

Features

GraphQL enables us to query the database by using a single REST endpoint. This application capitalizes on this feature to connect to MongoDB database, reducing the network bandwidth requirement. Apollo provider and Apollo Store makes React components compatible with GraphQL. Mutations and Query help to interact with the database.

Database

The database is hosted on mLab: MongoDB database. The app uses the Schema Definition Language to query or modify the database. In order to improve the performance of Like button, Caching mechanism is used to prevent calculating the same path in the graph again and again. Apollo Client aids in making the application data agnostic.
