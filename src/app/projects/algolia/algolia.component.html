Algolia Search

Angular based app that is based on Algolia for the restaurant dataset.

General

This project uses Algolia based Angular components to search through the Restaurants. The app has the ability to add or delete a restaurant. It can sort the list based on Star Rating and cuisines.

Technology

It uses Angular, Angular Material, SCSS, Algolia and Typescript. It is deployed using Vercel.

Algolia

Algolia enables fast searches on datasets that are uploaded on their platform. It then exposes certain components and APIs to interact with the dataset. It was a pretty cool experience building this app.

CDN

The CDN integration enables the users to upload their restaurants image to the dataset, which they can then search for.

Features

The stars are dynamic and are based on the rating of the restaurant. The CSS transitions, hover effects, sticky left panel and sticky search box enhances the user experience.
