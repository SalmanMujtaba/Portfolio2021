Node Based Chat App

The chat app is based on Node’s Socket.io which enables real-time bidirectional event-based communication. It allows Users to chat with friends and even share their location via the app. Check out the app via this link.

Features

The app enables a User to send or receive messages instantly. It uses Node’s Socket.IO, which allows the user to subscribe (join) to a chat room. All the online users of the chat room could be seen in the sidebar. Socket methods such as socket.on, socket.emit and io.emit, along with callbacks, facilitates in message exchange. The user can also send their location to the chat room.

Configuration

Node packages used: Express, Moment, Nodemon and Socket.IO.

Styling

Styled the app using Bootstrap. User scroll is tracked so that new messages are displayed to the user effectively.

Code

This app is synthesized using vanilla javascript, jQuery, HTML, CSS and bootstrap. Wrote several unit test cases using Mocha framework. Some front end stuff is crafted using Handlebars.
