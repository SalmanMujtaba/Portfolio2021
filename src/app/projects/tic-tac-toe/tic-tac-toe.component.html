Tic-Tac-Toe

A simple Tic-Tac-Toe game based on Angular, HTML and SCSS.

HTML

The project helped to understand the 2D matrix by creating this familiar Tic-Tac-Toe game. The UI is responsive and built using SCSS and HTML.

Angular

This is an Angular based project where the result of the game is determined as the users play their turn. An overlay at the end displays the final result of the game.
