Trip Manager

This app enables the user to manage their trips. The User can not only add a Trip, but he/she can add nearby places to the trip. Moreover, one can view all the selected places on a Google Map. The data persists even after the user closes the app.

Features

Trip Manager enables a User to add Trips to their account. The app provides suggestion to the User when he/she types a place to search. The app also permits the user to add nearby places to the previously added place. Google Map integration entitles the user to view all the added places of a Trip on a Map containing markers.

Code

The idiosyncrasy of this app is the Parent-Child relationship in a single Recycler View. A REST API is hit to implement the autocomplete functionality. Another API call fetches the nearby places of the previously selected selected place. Yet another API call fetches the data for getting the coordinates of all the selected locations, to view them on a Map.

Libraries

Picasso, okhttp, Firebase related libraries, Expandable Recycler View.
